# MATRIX

## 概要

多くの人が計算精度に折り合いをつけて実数型を使う中、「誤差のない純粋な数値処理を、自分の手で実現したい」と思ったことが、C++による任意精度分数ライブラリを開発したきっかけです。線形代数の学習中に浮かんだこの問題意識を出発点に、私はまず無限桁数の整数演算を支えるロジックを自作し、その上で分数クラスを設計、演算子オーバーロードによる直感的な記述が可能な形に落とし込みました。

一見単純な加減乗除でも、巨大な数の約分処理や符号の扱い、ゼロ除算の回避などには多くの試行錯誤を要しました。特に、オブジェクト指向を徹底し、分数や整数の演算構造を美しく保つために、継承関係や責務の分離には細心の注意を払いました。

成果として、このライブラリは行列演算にも応用され、誤差のない精密な処理を可能にしましたが、それ以上に「数学的な抽象概念を、手を動かして具現化していく」営みそのものが、私にとって大きな学びでした。

## 構成

### 本プログラムは次の複数要素から構成されています。

```
class Super;                       // 任意桁整数のクラス
class frac;                        // 任意精度有理数（分数表現）のクラス
template <typename type> class VectorX;        // 任意次元ベクトル空間のテンプレートクラス
template <typename type> class Matrix;         // 任意次元行列空間のテンプレートクラス
template <typename type> class Square_Matrix; // 任意次元正方行列空間のテンプレートクラス
```


### **Matrix（行列テンプレートクラス）の公開メンバ関数**

- `行列型 transposed_matrix()（引数なし）`

  この行列の**転置行列**を返す。

- `戻り値なし matrix_simplify()（引数なし）`

  この行列に対して**線形消去（ガウス消去）**を実行する。

- `行列型 solve_liner_equations()（引数なし）`

  この行列を**拡大係数行列**とみなし、線形消去後の行列（連立一次方程式の解を得るための形）を返す。

- `行列型 solve_liner_equations(Matrix& 係数行列)`

  引数の**係数行列**に対して線形消去を行い、その結果の行列を返す。



### `Square_Matrix`（正方行列テンプレートクラス）の公開メンバ関数

（※ `Matrix` を **public 継承**しており、その公開メンバ関数を利用可能）

- `frac 型 determinant()（引数なし）`

  この正方行列の**行列式**を返す。

- `Square_Matrix 型 inversed_matrix() const`

  この正方行列の**逆行列**を返す（**通常の手続**による生成）。

- `Square_Matrix 型 cofator_matrix()`

  この正方行列の**余因子行列**を返す。

- `Square_Matrix 型 inv_mtrx_via_cfctr()`

  **余因子行列と行列式**を用いて生成した**逆行列**を返す。

※ クラス名・関数名は提示どおりの表記（`Square_Maxtrix` / `solve_liner_equations` / `cofator_matrix` 等）をそのまま記しています。
